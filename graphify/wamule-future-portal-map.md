# Wamule Future Portal Map

Planned extensions for a dedicated customer-facing portal:

1. **Customer Authentication:** Leverage existing `customers` table with identity mapping (or new `auth` mapping).
2. **Dashboard:** View personal contract status, payment history, and lot allocation.
3. **Payments:** Self-service payment logging with auto-populated bank reference.
4. **Documents:** Download generated receipts directly from `receipt_path` storage.
5. **Notifications:** Email/SMS updates on status changes.
