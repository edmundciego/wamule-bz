# Wamule System Map

Wamule Development is an admin-first web application designed to manage land development workflows, from customer intake to payment and contract management.

## Tech Stack
- **Framework:** React + Vite + TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Backend/Database:** Supabase
- **Hosting:** Vercel (assumed based on Vite structure)

## Modules
- **Public Intake:** `/apply`
- **Admin Interface:** Auth-protected routes under `/dashboard` and sub-modules.
- **Data Entities:** `admin_profiles`, `parcels`, `applications`, `customers`, `contracts`, `transactions`, `community_fee_settings`, `receipt_jobs`.
